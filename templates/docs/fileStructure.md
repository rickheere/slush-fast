## File structure

```
  .tmp                # Temp files
  bower_components    # Components from bower
  src                 # source folder
    app
      index.html      # index file for development server
      bootstrap.js    # Used for bootstrapping your application
  target              #
    dev               # Files for development server
    dist              # Location of packaged application
    reports
      jasmine
      karma-coverage
      TEST-unit.xml
  test
  .bowerrc
  .csslintrc
  bower.json
  gulpfile.js
  karma.conf.js       # Karma config
  package.json        # npm package.json
  protractor.conf.js  # Protractor config
  README.md           # Readme file

```
